Evaluation

In Video Game Development, we used the software GDevelop, a code-free, simple game engine that anyone can use. I learned a lot of skills within that specific software and I enjoyed it. The only downsides I encountered were the outdated tutorials and the limited time we had to create the game. I could definitely extend this by making different kinds of games beyond just platformers and using software that involves more complex coding.

I believe that creating, designing, and developing a video game is something I can consider for my final project.
